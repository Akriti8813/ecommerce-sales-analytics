# E-commerce Sales Analytics Dashboard

This project analyzes e-commerce transaction data using SQL, Python, and Power BI.

## Tools Used
Python (Pandas)
SQL
Power BI

## Analysis Performed
- Revenue trend analysis
- Product performance analysis
- Customer segmentation
- Seasonal demand analysis

## Dashboard Metrics
- Total Revenue
- Average Order Value
- Monthly Sales Trend
- Top Products

Dataset: Brazilian E-commerce dataset (100k+ transactions)
